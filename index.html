Build your UI without leaving HTML

Twippo is a CSS Framework with classes like p-xl, rotX-90 or f-poppins that help you to build your UI faster.

Get Started

<link rel="stylesheet" href="https://www.combinatronics.com/Twippolabs/Twippo/main/source/twippo.css"/>
@import url("https://www.combinatronics.com/Twippolabs/Twippo/main/source/twippo.css");
https://github.com/Twippolabs/Twippo/archive/refs/heads/main.zip

Code requirements

Use the HTML5 doctype

<!DOCTYPE html>

Add the responsive viewport meta tag

<meta name="viewport" content="width=device-width, initial-scale">

Starter template

If you want to get started right away, you can use this HTML starter template. Just copy/paste this code in a file and save it on your computer.

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale">
<title>Twippo</title>
<link href="https://www.combinatronics.com/Twippolabs/Twippo/main/source/twippo.css"/>
</head>
</body>
<div class="section">My first website with Twippo!!!</div>
</body>
</html>

Help you to build UI faster

With short class, but many css, Twippo allows you to save about 60% time creating an UI.
